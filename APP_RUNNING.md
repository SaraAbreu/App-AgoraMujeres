## 📵 Cómo Acceder a la App

### Opción 1: Web en Navegador
1. Abre tu navegador
2. Ve a: **http://localhost:8082**
3. ¡La app estará funcionando!

### Opción 2: Emulador/Dispositivo Real
- Abre la terminal del Frontend
- Presiona `W` para web
- Presiona `A` para Android (si tienes emulador)
- Escanea el QR con Expo Go (en tu teléfono)

### Opción 3: Probar el Chat Directamente
En una terminal ejecuta:
```bash
.\.venv\Scripts\python.exe test_chat_openai.py
```
